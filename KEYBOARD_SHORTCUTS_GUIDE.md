# MediMind EMR - Keyboard Shortcuts Guide

Quick reference for power users to speed up workflows.

---

## Patient Registration Form

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+Enter** (Windows/Linux) | Submit Form | Save patient registration |
| **Cmd+Enter** (Mac) | Submit Form | Save patient registration |
| **Escape** | Cancel | Close form without saving |
| **Tab** | Next Field | Move to next input field |
| **Shift+Tab** | Previous Field | Move to previous input field |

### Usage Tips
- Shortcuts work from any field in the form
- Submit is disabled during form validation
- Cancel only works if close handler is available

---

## Patient Search

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Enter** | Search | Execute patient search |
| **Escape** | Clear | Clear all search filters |
| **Tab** | Next Filter | Move to next search field |

### Usage Tips
- Press Enter in any search field to start search
- Escape clears all filters instantly
- Search results appear below form

## Troubleshooting

**Shortcut not working?**
1. Check if another application is using the same shortcut
2. Try refreshing the page (Ctrl+R / Cmd+R)
3. Ensure you're focused on the correct form/view
4. Check browser console for errors

**Shortcuts conflicting with browser?**
- Some browsers reserve Ctrl+S (Save page)
- MediMind avoids common browser shortcuts
- Use provided shortcuts instead
