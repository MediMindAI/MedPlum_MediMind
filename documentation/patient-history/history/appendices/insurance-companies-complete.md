## Usage Notes

1. **Value "0"** or empty string = "შიდა" (Internal) - default/no external insurance
2. **Value "14137"** = "დაზღვევის არ მქონე" (No Insurance) - patient has no coverage
3. **Government payers** include values: 628, 7603, 9155, 10483, 61677, 70867
4. **Private insurance companies** include values: 6379-6385, 8175, 11209, 16476, 16803, 41288, 46299, etc.
5. **Medical institutions** acting as payers: 49974, 67209, 79541, 81614, 86705, 88950, 89213, 89718, 91685
