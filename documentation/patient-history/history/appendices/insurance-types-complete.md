## Program Categories

### Social Protection Programs
- **Value 10**: საპენსიო (Pension) - for pensioners
- **Value 13**: პედაგოგი (Teacher) - for educators
- **Value 14**: უმწეო (Vulnerable/Poor) - social assistance
- **Value 15**: შშმპ (Persons with Disabilities)
- **Value 33**: ვეტერანი (Veteran)

### Universal Healthcare Programs (Code 36)
Basic state healthcare coverage with different tiers:
- **Values 17-21**: Original 36 program (Minimal & Standard)
- **Values 41-46**: New 36 program variants

### Point-Based Programs (Codes 165, 218)
Insurance programs with point-based coverage limits:
- **165 series** (values 22-25, 57): 165-point programs
- **218 series** (values 26-29): 218-point programs
- **Values 39-40**: Basic < 1000 / >= 1000 points
- **Values 50-51**: Minimal < 1000 / >= 1000 points
- **Value 49**: 70000-100000 point range

### Corporate & Municipal Programs
- **Value 30**: კორპორატიული (Corporate insurance)
- **Value 36**: Tbilisi Mayor's Medical Service

### Specialized Medical Programs
- **Value 32**: State Referral Service Program
- **Value 37**: Vascular Approach Coverage
- **Value 60**: Oncology Component with Co-payment
- **Value 61**: Chemotherapy and Hormone Therapy
- **Value 62**: Heart Surgery Additional Services

### DRG (Diagnosis Related Group) Programs
- **Value 58**: DRG Planned Surgery
- **Value 59**: DRG Emergency Stationary

### Refugee/Displacement Programs
- **Value 63**: IDPs & Eco-migrants Agency

## Program Number System

**Program Code Structure**: `[Base]-[Component]-[Description] ([Variant])`

### Base Codes:
- **36**: Universal Healthcare Program
- **165**: 165-point coverage program
- **218**: 218-point coverage program

### Component Codes:
- **2**: Emergency Ambulatory
- **3**: Emergency Stationary
- **4**: Planned Surgical Service
- **5**: Cardiac Surgery

### Variants:
- **მინიმალური** (Minimal): Lower coverage tier
- **ახალი** (New): Updated program versions (post-2017)

---

## Historical Context

**2017 Reform**: Programs with "(ახალი)" suffix were introduced after January 1, 2017, with updated coverage rules (see values 52-53).

**Point System**: Earlier programs used point-based limits (165, 218, 1000, 40000, 70000-100000) to define maximum coverage amounts.

**DRG Introduction**: Newer programs (values 58-59) use Diagnosis Related Group (DRG) methodology for payment calculation.

---

## Usage Notes

1. **Value "0"** or empty = No specific program selected
2. **Value "38"** ("-") = Placeholder/undefined
3. Insurance companies may support only specific program types
4. Some programs have age restrictions (e.g., value 56: ages 6-18)
5. Co-payment percentages vary by program type
